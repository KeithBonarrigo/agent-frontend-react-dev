# BotWerx Demo Script

> **Target audience:** Real estate professionals interested in listing individual properties via social media, promoting them across channels, and using audience segmentation to target buyers.
>
> **Estimated runtime:** 25–30 minutes

---

## Pre-Demo Checklist

- [ ] Test agent configured (EasyBroker or MLS level)
- [ ] WhatsApp, Messenger, and/or Instagram integrations connected
- [ ] Sample conversations and leads in the system
- [ ] At least one Audience Insights report generated
- [ ] At least one Income Report generated
- [ ] Chatbot embedded on a test domain

---

## Act 1: The Hook (2 min)

**"What if your property listings could promote themselves — across every channel — 24/7?"**

### Home Page
- [ ] Show the hero section: zero code, multi-channel, insights, security
- [ ] Point out the signup form — getting started takes minutes

### Agent Page — Live Chatbot Demo
- [ ] Click the chat launcher, ask it about a property
- [ ] Show how the bot answers with listing details, photos, pricing
- [ ] "This is what your buyers see — instant, intelligent answers about your listings"

---

## Act 2: Setting Up Your Agent (5 min)

### Agent Settings
- [ ] Show agent name, company, domain, contact info
- [ ] Highlight timezone and response time restriction
- [ ] "Your bot works after hours so you don't have to"

### Configurations → Instructions
- [ ] Show how to add custom prompts that shape the bot's personality
- [ ] Demo: _"Always highlight the property's neighborhood amenities"_
- [ ] Drag-and-drop reordering of instructions

### Configurations → Training (key for property listings)
- [ ] **URL Embeddings**: Submit a property listing page URL — the bot learns it
- [ ] **File Embeddings**: Upload a property brochure PDF
- [ ] "Your bot now knows everything about this listing — price, features, location, photos"
- [ ] Show the list of trained URLs/documents

### Configurations → Models
- [ ] Show model selection (Claude / GPT) based on subscription level

### Styling & Embed
- [ ] Show branding customization: logo, bot avatar, greeting message
- [ ] Show the embed code — one script tag on your website
- [ ] Click "Test Load" to show it working live

---

## Act 3: Multi-Channel Property Promotion (5 min)

**"One agent, every channel your buyers are on."**

### Website Embed
- [ ] Show the embed snippet and copy button
- [ ] "This is your 24/7 listing agent on your website"

### WhatsApp Integration
- [ ] Show connected WhatsApp number
- [ ] "Buyers text your number, the bot answers with property details instantly"
- [ ] Walk through the verification flow (phone → SMS code → PIN)

### Facebook Messenger
- [ ] Show connected Facebook page
- [ ] "You post a listing on Facebook, someone clicks 'Message' — the bot handles it"
- [ ] "No more missed inquiries on your Facebook listings"

### Instagram DMs
- [ ] Show connected Instagram account
- [ ] "You post a property photo on Instagram, someone DMs you — handled automatically"
- [ ] Demo the test message feature
- [ ] "Every Instagram post becomes a lead capture opportunity"

### Google Calendar
- [ ] Show connected calendar status
- [ ] "When a buyer wants to schedule a showing, the bot checks your real availability"
- [ ] "No more double-bookings, no more back-and-forth"

## Act 4: Lead Capture & Sharing (5 min)

**"Every conversation is a potential lead."**

### Conversations Tab
- [ ] Show the conversation list grouped by domain and channel
- [ ] Expand a conversation — show the full chat thread
- [ ] Point out captured contact info (name, email, phone, location)
- [ ] Show the "Live" indicator for active sessions
- [ ] "You can see exactly which listings are generating conversations"

### Live Map
- [ ] Expand the Live Map accordion
- [ ] "See exactly where your interested buyers are — in real time"
- [ ] Click a marker to jump to that user's conversation
- [ ] "This tells you which neighborhoods have demand"

### Leads Tab
- [ ] Show the leads list with count badge

### Share & Invite (key for social promotion)
- [ ] **Email Sharing**: Enter a colleague's email, customize the message, send
- [ ] "Share a hot lead with your team in seconds"

- [ ] "Send this link to a buyer group on WhatsApp — they click, they're chatting with your bot about the property"

### Lead Searches & Scheduling
- [ ] Show saved search queries

- [ ] "Automatically re-run searches to catch new interested buyers"

### Past Reports

### Metrics Dashboard

- [ ] Conversation count with quick-link
- [ ] Lead count with quick-link

- [ ] "Your buyers are concentrated in these areas — focus your listings there"

## Key Talking Points for Property/Social Media Audience

1. **"Train it on any listing"** — paste a URL or upload a PDF, the bot becomes an expert on that property
2. **"Every social channel is a storefront"** — Instagram post → DM → bot answers → lead captured
3. **"WhatsApp links are shareable ads"** — generate a wa.me link, post it anywhere, buyers click and chat
4. **"Segmentation drives strategy"** — Audience Insights tells you which property types resonate on which channels
5. **"Real-time demand map"** — Live Map shows where buyers are geographically concentrated
6. **"Showings book themselves"** — Google Calendar integration means the bot schedules around your availability
7. **"Scale per listing or per market"** — create separate agents for different property types or neighborhoods
